Vehicle Repair is a web application that lets you keep track of vehicle maintenance, 
services  and store information such as vehicle owner, make, model, year, engine type 
and more for each vehicle. Also, application lets you keep track of all your customer data,
 such as contact information.
Vehicle Repair web application is intended to anyone who works in vehicle repair industry.
Technologies used: ASP.NET Web Forms, C#, Html, CSS.
